# My React Portfolio

A simple personal portfolio built with React + Vite.

## Scripts

- `npm run dev`: start dev server
- `npm run build`: production build
- `npm run preview`: preview built app

## Getting Started

```bash
cd portfolio
npm install
npm run dev
```

Then open the local URL printed in the terminal (usually http://localhost:5173).

## Customize
- Update `src/pages/*` with your details.
- Edit colors/layout in `src/styles.css`.
- Add projects to `src/pages/Projects.jsx`.
